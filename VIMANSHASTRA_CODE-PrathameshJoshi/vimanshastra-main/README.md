**From Viman Shastra to Modular Safety**

Welcome to **Wings of Time**, an interactive exploration bridging the ancient aviation wisdom of the Vimanika Shastra with modern aerospace engineering. This project features a deep dive into **Kinetic Mass Reduction (KMR)**, showcasing a conceptual modular aircraft safety system designed to dramatically reduce crash impact energy by safely jettisoning non-essential mass.

## 🚀 Key Features

* **Interactive Modular Schematic:** A responsive, interactive dashboard highlighting the Fail-Safe Protocol matrix, where users can interact with an aircraft's ejection system.
* **Modern Aesthetic:** Features a custom sleek, dark theme with glassmorphism UI cards, glowing accents, and seamless high-tech typography.
* **Ancient vs. Modern:** Explores the physical and theoretical translations from the ancient Indian text "Vimanika Shastra" into practical aerospace blueprints.
* **Calculated Safety:** Demonstrates the conceptual math behind Kinetic Mass Reduction, estimating the theoretical impact energy reduction.
* **Fully Responsive:** Beautifully crafted layout to ensure accessibility across all devices.

## 🏁 Getting Started Locally

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed previously on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Vasundhara1819/vimanshastra.git
   cd vimanshastra
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run start
   ```

4. **View the App:** Open your browser and navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

---

## 📦 Building and Deploying

To create a production-ready build of this project, run:
```bash
npm run build
```
This compilation step will package and optimize the application into the `dist/` directory.

### Deploying to Firebase

If you want to manually deploy to the live server:
```bash
npx firebase-tools deploy --only hosting
```
